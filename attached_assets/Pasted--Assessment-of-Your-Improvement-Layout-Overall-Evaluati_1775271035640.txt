## What My Plan Adds That You Should Consider

### 1. **Race Condition Prevention**
**Your plan doesn't address**: Two hourly triggers (master sync + shared refresh) running independently.

**Solution**: In your Apps Script, make the master sync **call** the shared list refresh as its final step. Remove the separate trigger on the shared list script.

```javascript
// At end of master sync script:
function onSyncComplete() {
  try {
    // Trigger shared list refresh
    var sharedScript = ScriptApp.newTrigger('refreshSharedList')
      .timeBased()
      .after(1000) // 1 second delay
      .create();
  } catch (e) {
    Logger.log('Failed to trigger shared refresh: ' + e);
  }
}
```

### 2. **"NOT FOUND" Retry Logic**
**Your plan doesn't address**: Once Carfax/Website/Price is marked "NOT FOUND", it stays that way forever.

**Solution**: Add a "Last Attempted" timestamp column. In your Carfax workflow, skip "NOT FOUND" entries unless they're >24 hours old.

```javascript
// In Carfax automation:
if (carfaxValue === 'NOT FOUND') {
  const lastAttempt = new Date(row.lastAttemptedTimestamp);
  const hoursSince = (Date.now() - lastAttempt) / (1000 * 60 * 60);
  if (hoursSince < 24) continue; // Skip, try again tomorrow
}
```

### 3. **Input Validation on doPost**
**Your plan mentions**: API key check (good!)

**Also add**: Validate the data itself to prevent corruption:

```javascript
function doPost(e) {
  // Check API key
  if (e.parameter.apiKey !== PropertiesService.getScriptProperties().getProperty('API_KEY')) {
    return ContentService.createTextOutput(JSON.stringify({error: 'Unauthorized'}))
      .setMimeType(ContentService.MimeType.JSON);
  }
  
  // Validate row index
  const rowIndex = parseInt(e.parameter.rowIndex);
  if (isNaN(rowIndex) || rowIndex < 2) {
    return ContentService.createTextOutput(JSON.stringify({error: 'Invalid row index'}))
      .setMimeType(ContentService.MimeType.JSON);
  }
  
  // Validate Carfax value format
  const carfaxValue = e.parameter.carfaxValue;
  if (!carfaxValue.startsWith('http') && carfaxValue !== 'NOT FOUND') {
    return ContentService.createTextOutput(JSON.stringify({error: 'Invalid Carfax value'}))
      .setMimeType(ContentService.MimeType.JSON);
  }
  
  // Write to sheet...
}
```
