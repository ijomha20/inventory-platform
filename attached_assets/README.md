# Attached Assets

Captured API payloads, reference documents, and automation scripts collected during development.

These are **not live code**. They are snapshots of CreditApp GraphQL responses, worksheet data,
and other reference material used for debugging and development context.

Contents include:
- `Pasted--data-application-*` — CreditApp application/worksheet API response captures
- `Pasted--data-currentUserRole-*` — CreditApp user role/retailer context captures
- `Pasted--data-updateCreditorApplicationWorksheetDraft-*` — Worksheet draft mutation responses
- `carfax-automation/` — Reference Carfax automation scripts and documentation
- `*.gs` — Google Apps Script source files

Do not use these files as the source of truth for lender parameters — they are point-in-time
captures that may not match the current live program configuration.
